== Description ==

With this plug-in you can integrate your WordPress website with any of ContentSuit features.
The integration is very simple. The plug-in is responsible for adding the TAGs of all features inside your WordPress without you having to change your current code. 
You can check more details on the <a href="https://www.contentsuit.com/how-it-works">How it works</a> page.
When you enter the key you received by email. The plug-in will connect with the ContentSuit servers and bring in all the features you have chosen.
You can <a href="https://www.contentsuit.com/features">choose new features</a> from the ContentSuit website. Feel free to change everything!

== Installation ==

= Automatic installation =
1. Go to Plugins > Add New > Upload and select the .zip file from your hard disk
2. Click the "Install now" button
3. Activate the plug-in through the 'Plugins' menu in your WordPress
4. Find "ContentSuit" option into the 'Settings' menu in your WordPress to use

= Manual installation =
1. Upload the plug-in folder to the /wp-content/plugins/ directory via ftp
2. Activate the "ContentSuit" plug-in through the 'Plugins' menu in WordPress
3. Find "ContentSuit" option into the 'Settings' menu in your WordPress to use

== Frequently Asked Questions ==

= Where do features come from? = 

All features you choose are hosted on ContentSuit servers. This way, you gain agility when requesting any changes.

= How can I get the key ID? =

Your key ID will be sent to you by email after you choose the features you want.

= My style is not applied =

Verify if the TAGs for the features have been found. If yes, check the ContentSuit Customer Dashboard if features are already enabled.
